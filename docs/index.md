## 워크북 사용법

이 워크북은 **직접 따라하며 배우는 실습 중심** 자료입니다.

- **[개념]** → 읽고 이해하는 파트
- **[실습]** → 직접 수행하는 파트
- **[체크포인트]** → 진행 상황 확인
- 막히면 언제든 질문하세요!

> **소요시간:** 약 3~4시간 (1부 + 2부)
> **사전 준비물:** 노트북, 인터넷 연결, Claude Pro 또는 Max 구독
> 코딩 경험 불필요. 터미널 경험 불필요.
