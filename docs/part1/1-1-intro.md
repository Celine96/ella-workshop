## 1.1 클로드 코드란?

### [개념] 클로드 코드 = 내 컴퓨터에서 직접 일하는 AI 비서

일반적인 AI 챗봇(ChatGPT 등)은 **대화만** 합니다.
클로드 코드는 다릅니다. 내 컴퓨터의 파일을 직접 읽고, 만들고, 수정하고, 명령을 실행합니다.

알아야 할 명령어는 딱 하나: `claude`

### [개념] ChatGPT와 뭐가 다른가요?

| 기능 | ChatGPT | 클로드 코드 |
|------|---------|-----------|
| 파일 접근 | 불가 | 내 컴퓨터 파일 읽기/쓰기 가능 |
| 기억력 | 대화 끝나면 사라짐 | CLAUDE.md로 영구 기억 |
| 실행력 | 조언만 해줌 | 직접 실행함 |
| 멀티 파일 | 한 번에 하나 | 수십 개 동시 처리 |
| 에이전트 | 단일 인스턴스 | 병렬 처리 가능 |

> **핵심 차이:** ChatGPT는 조언을 해주지만, 클로드 코드는 **직접 일을 합니다**.

### [개념] 마인드셋 전환

```
기존 사고방식:  "명령어를 배워야 해"  →  코딩, 기술 학습에 집중
새로운 사고방식: "결과물이 필요해"    →  원하는 결과를 말하면 클로드가 방법을 찾음
```

여러분은 **오케스트레이터(지휘자)** 입니다. 코딩을 배우는 게 아니라, AI에게 원하는 결과를 전달하는 방법을 배웁니다.

### [개념] 준비물

- Claude Pro 또는 Max 구독 ($20/월~)
- Windows / Mac / Linux 컴퓨터
- 호기심과 열린 마음 :)
