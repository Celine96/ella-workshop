1.4 실습 1: 카드뉴스 생성 — Step 1을 시작합니다.

실거래가 CSV 데이터를 읽고, 인스타그램 카드뉴스 PNG 이미지를 만들어 볼게요.

이 폴더에 있는 실거래가 CSV 파일을 읽고, 사전 공유된 templates 폴더의 HTML 템플릿을 참고해서 인스타그램 카드뉴스 PNG 이미지를 만들어줘.

데이터 선별 기준:
- 유형이 "일반"인 것만 (상가/집합 제외)
- 표제부 매칭이 성공한 것만 (지번 주소가 확보된 데이터)
- 2026년 계약 건만
- 최고가 1건 + 최저가 1건 (최저가는 1억 이상)

슬라이드 구성:
- 1장: 표지 (제목 + 날짜)
- 2장: 최고가 거래 데이터 (주소, 거래금액, 면적, 용도, 건축년도)
- 3장: 최저가 거래 데이터

디자인 사양:
- 크기: 1080x1080px
- 배경: 건물 사진 + 그라데이션 오버레이
- 강조색: 개나리 노란색 (#FFD700)
- 텍스트: 흰색, 깔끔한 산세리프
- 데이터 테이블: 우측 하단 배치
- 파일명: 지번 주소로 저장 (예: slide_2_동교동 160-5번지.png)

생성이 완료되면 다음 단계로 넘어가 주세요: /1-4-practice1-step2
