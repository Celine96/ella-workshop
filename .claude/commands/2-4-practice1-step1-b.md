B. 데이터 + 디자인 파이프라인을 선택하셨군요!

데이터를 가져와서 자동으로 디자인 결과물을 생성하는 파이프라인을 만들어 볼게요.
1.4에서 체험한 카드뉴스 파이프라인과 비슷한 구조예요.

아래 질문에 답변해 주세요:
1. 어떤 데이터를 사용하나요? (예: 엑셀, CSV, API 등)
2. 최종 결과물은 어떤 형태인가요? (예: 이미지, PDF, 문서 등)
3. 데이터에서 결과물까지 현재 어떤 단계를 거치나요?
4. 가장 자동화하고 싶은 단계는 뭔가요?

답변을 바탕으로 파이프라인을 설계해 드리겠습니다.
