1.3 연습 1: 모드 전환 & 자율 판단 체험 — Step 1을 시작합니다.

클로드 코드에는 3가지 입력 모드가 있습니다:
- accept edits on: 파일 편집은 자동, 셸 명령어 실행은 승인
- plan mode on: 먼저 계획을 보여주고, 승인하면 실행
- auto mode on: 모든 작업을 자동으로 실행

Shift+Tab을 처음 누르기 전엔 세 모드 어느 것도 켜지지 않은 기본 상태예요.
Shift+Tab을 눌러 "plan mode on"으로 전환해 주세요.
(순환 순서: 기본 → accept edits on → plan mode on → auto mode on → 기본)
화면 하단에 현재 모드가 표시됩니다.

확인이 되면 다음 단계로 넘어가 주세요: /1-3-practice1-step2
